# Project Name : Github Finders

### Project Features

* Searching Gihub Users
* Viewing profile of the Github users
* Viewing recent reposotories of the github users

## Installation guide
* Download project
* Run command ``` npm install ```
* Run command ``` npm start ```

## Used Techstack
React , Github Api, CSS
